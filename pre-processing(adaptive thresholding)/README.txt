Topic : Pre-processing of Degraded documents

Introduction :

The objective of the project is to pre process the degraded documents so that it is easily recognisable by OCR.

Requirements:

python3 is required to run the file
following library is needed to be installed:
CV2,matplotlib,numpy

How to execute application:

run katta2.py in the file to execute the program.

original images are saved in test file and processed images are saved in test_cleaned file in the form of same name as the original file name.

The global and adaptive tresholding both are done for input image .
